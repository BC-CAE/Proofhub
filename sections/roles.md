Get roles
----------------

* `GET v3/roles` will return all account roles.

```json
[
    {
        "id":24903232,
        "name":"Owner",
        "default":true
    },
    {
        "id":24903236,
        "name":"Admin",
        "default":true
    },
    {
        "id":249032326,
        "name":"Normal User",
        "default":true
    },
    {
        "id":24903237,
        "name":"Client",
        "default":false
    }
]
```
